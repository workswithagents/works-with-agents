# AGENTS.md — workswithagents.com

Global education platform. Teaches the 10-pattern agentic methodology.
Co-founded by Vilius Vystartas (technical) & Pelin Kayhan (business/compliance).

## What this is

Public-facing site converting visitors from "what are AI agents" to "I want to learn this properly."

## Pages

- **Home** — hero, methodology overview, 10 patterns
- **Learn** — full course outline, 10 module descriptions, learning formats
- **Newsletter** — signup form, value proposition, connected to API
- **Blog** — practical writing on agentic methodology (coming soon)
- **Contact** — email contacts, cross-domain links
- **About** — founders, mission, what we actually do
- **llms.txt** — AI agent documentation index (llmstxt.org standard)
- **llms-full.txt** — full education docs for AI agents

## Differentiation

- Not "AI will change everything" — "here's a specific pattern, when it works, when it doesn't"
- Developer-focused, not executive-focused
- Real examples from autonomous agent experimentation
