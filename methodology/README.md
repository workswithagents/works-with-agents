## Learning Path

**Phase 1: Foundation** (Patterns 1-3)
Boot → Skills → Memory. These three give your agent context, procedures, and durability. Start here.

**Phase 2: Autonomy** (Patterns 4-6)
Decision Protocols → Tool Composition → Orchestration. Your agent learns when to act, which tool to use, and how to parallelise.

**Phase 3: Scale** (Patterns 7-8)
Pipelines → Resilience. Your agent works on schedules, handles failures, and self-heals.

**Phase 4: Harden** (Patterns 9-10)
Verify → Compounding. Quality gates catch errors. The system becomes self-reinforcing.
